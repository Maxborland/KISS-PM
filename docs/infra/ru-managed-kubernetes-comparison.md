## Ключевые выводы

### Managed k3s в России не предлагает никто
Все семь провайдеров дают полноценный vanilla CNCF Kubernetes (containerd), а не k3s. Поэтому реальная развилка:

- **(A) Managed k8s у провайдера** — провайдер держит control plane (часто бесплатно), обновления, HA. ~12–33к ₽/мес, ops-нагрузка минимальна.
- **(B) Self-managed k3s на голых VPS/выделенных серверах** — k3s ставится самостоятельно на дешёвые VM. Железо дешевле, но весь ops на нас: HA control plane, бэкап etcd, обновления, сертификаты, мониторинг.

**Рекомендация для запуска:** не брать self-managed k3s на старте — экономия 5–10к ₽/мес не оправдывает операционного риска без выделенного DevOps. k3s оправдан позже — для edge / self-hosted enterprise-поставок клиенту.

### Рекомендация по провайдеру (под разные приоритеты)

1. **Selectel — основной выбор для прод-SaaS.** Бесплатный HA control plane, лучший SLA (99,98%) с внятной формулой компенсации, прозрачные цены, зрелая платформа, дешёвые диски, есть аттестованные сегменты под 152-ФЗ. ~20,5к ₽/мес.
2. **Timeweb — если приоритет минимальная цена.** В 1,5–2× дешевле (~12–14к) + бесплатный безлимитный egress (важно для медиа/LiveKit). Минусы: компенсации по SLA только зачётом, фиксированные конфиги нод, моложе как платформа.
3. **Cloud.ru / КРОК — если важен enterprise-бэкинг и реестр российского ПО** (госклиенты, крупные сделки). КРОК — бесплатный HA control plane, но цены непрозрачны (нужен запрос в sales).

### Предостережения перед договором
- **Selectel анонсировал изменение части цен с 01.07.2026** — перепроверить.
- Для медиа-плоскости (LiveKit/coturn — UDP-трафик, публичные порты) уточнить egress и доступность публичных UDP-портов. Здесь бесплатный трафик **Timeweb** и **VK Cloud** выигрывает.
- Для РФ-SaaS с персональными данными нужен провайдер с аттестованным сегментом под **152-ФЗ** — предлагают как отдельную услугу большинство крупных (Selectel, Cloud.ru, VK, Yandex, MWS).

## Детальные данные по провайдерам

### Timeweb Cloud — самый дешёвый

Продаёт ноды готовыми конфигурациями с фиксированной ценой/мес (не «vCPU/час + RAM/час»).

- **Control plane:** платный, отдельной нодой. Dev 180 ₽/мес (1 мастер, не HA, до 10 нод); Base 1 800 ₽; Custom (HA, 1 или 3 мастера) от 2 520 ₽. Для прод-HA нужен Custom с 3 мастерами (≈ 2 520–4 500 ₽/мес).
- **Worker-ноды (готовые тарифы):** 4 vCPU / 8 ГБ / 120 ГБ NVMe = **2 990 ₽/мес** [точно]. 8 vCPU / 16 ГБ ≈ 3 200–3 800 ₽ [оценка].
- **SLA:** маркетинг 99,98%; юридически в SLA-PDF закреплено **не менее 99,9% по году**. Компенсация: при 6+ сбоях/сутки или простое 6+ ч — зачёт стоимости тарифа за сутки (не деньгами).
- **Балансировщик:** от 135 ₽/мес (500 Мбит/с) / 225 ₽ (1 Гбит/с) / 675 ₽ (HA).
- **Публичный IPv4:** 180 ₽/мес (с 16.02.2026 тарифицируется отдельно на worker-нодах).
- **Egress:** бесплатный и безлимитный.
- **S3:** 2 ₽/ГБ/мес (стандарт); базовый пакет 250 ГБ = 639 ₽/мес.
- **k8s/k3s:** vanilla k8s (v1.26–1.36.1, управление на базе k0s); k3s нет.
- **Прод-HA итог:** ≈ **12 000–14 000 ₽/мес**.

### Selectel — лучший баланс цена/SLA

- **Control plane:** **бесплатный** для отказоустойчивого (3 мастера) и базового (1 мастер). Базовый — без SLA.
- **Worker-ноды:** vCPU ≈ 884 ₽/мес, RAM ≈ 292 ₽/мес за ГБ, SSD universal v2 ≈ 9,6 ₽/ГБ·мес. 4 vCPU/8 ГБ (+50 ГБ SSD) ≈ **6 349 ₽/мес**; 8 vCPU/16 ГБ ≈ 12 219 ₽.
- **SLA:** **99,98%** (отказоустойчивый кластер). Компенсация: 0,5% стоимости за каждые 30 мин простоя [полу-подтверждено].
- **Балансировщик:** ≈ 1 271 ₽/мес. **Публичный IP:** ≈ 182 ₽/мес.
- **Egress:** 3 ТБ/мес внешнего бесплатно, далее ~0,52 ₽/ГБ; внутренний (к S3) бесплатно.
- **S3:** стандарт ≈ 2,29 ₽/ГБ·мес; холодный ≈ 0,81 ₽ [оценка].
- **k8s/k3s:** vanilla k8s (1.33–1.35.x); k3s нет.
- **Прод-HA итог:** ≈ **20 500 ₽/мес**.
- ⚠️ Анонсировано изменение части цен с 01.07.2026.

Источники: docs.selectel.ru/managed-kubernetes/about/payment · selectel.ru/prices

### Cloud.ru (Evolution Managed Kubernetes) — экс-SberCloud

Цены тарифа без НДС (НДС 22% сверху). Тариф №7.EVO.2 действует с 01.01.2026.

- **Control plane:** платный, бандлами VM. HA = 3 узла в разных зонах. 3× мастер 2 vCPU/4 ГБ ≈ **6 519 ₽/мес с НДС**.
- **Worker-ноды (100% vCPU):** 4 vCPU/8 ГБ = 4,87 ₽/час без НДС = **4 336 ₽/мес с НДС**; 8 vCPU/16 ГБ = **8 672 ₽/мес с НДС**. Есть доли vCPU 10/30/100%.
- **SLA:** **99,95%** для всех сервисов EVO. Компенсация: снижение платежа в следующем периоде, потолок ≤50% стоимости за период.
- **Балансировщик L4:** ≈ 586 ₽/мес с НДС + трафик. **Публичный IP:** ≈ 151 ₽/мес. **SSD NVMe:** ≈ 11,58 ₽/ГБ·мес.
- **Egress:** ~3,75 ₽/ГБ с НДС (без явного free-объёма — уточнять).
- **S3:** однозонный стандарт ≈ 1,13 ₽/ГБ·мес с НДС; ледяной ≈ 0,49 ₽.
- **k8s/k3s:** vanilla k8s; k3s нет.
- **Прод-HA итог:** ≈ **21 400 ₽/мес с НДС** (≈ 17 500 без НДС).

Источники: cloud.ru/documents/tariffs/evolution/managed-kubernetes · /load-balancer · /object-storage · /evolution-compute · cloud.ru/documents/contracts/terms-of-service/evolution/service-level-agreement

### Yandex Cloud — Managed Service for Kubernetes

- **Control plane (новая модель с 18.06.2026):** базовая ставка мастера = 0, платите за его vCPU (1,76 ₽/час) и RAM (0,46 ₽/час, с НДС); размер мастера Yandex масштабирует сам. Региональный (HA) мастер ≈ 7 826 ₽/мес [оценка]; зональный ≈ 3 913 ₽ [оценка].
- **Worker-ноды (100% vCPU, Cascade Lake):** vCPU 1,41 ₽/час, RAM 0,44 ₽/час. 4 vCPU/8 ГБ ≈ **6 687 ₽/мес**; 8 vCPU/16 ГБ ≈ 13 374 ₽.
- **SLA:** региональный мастер **99,90%**; зональный мастер — **SLA не предоставляется**. Компенсация: 10% (95–99,9%), 25% (<95%).
- **Балансировщик (NLB):** ≈ 672 ₽/мес + трафик. **Публичный IP:** ≈ 192 ₽/мес. **SSD:** ≈ 10,73 ₽/ГБ·мес.
- **Egress:** первые 100 ГБ/мес бесплатно, далее 1,42 ₽/ГБ.
- **S3:** стандарт ≈ 2,41 ₽/ГБ·мес; cold 1,28 ₽; ice 0,64 ₽.
- **k8s/k3s:** vanilla k8s (CNCF); k3s нет.
- **Прод-HA итог:** ≈ **29 800 ₽/мес с НДС**.

### КРОК / K2 Cloud — бесплатный HA control plane, но цены непрозрачны

С ноября 2024 платформа «Облако КРОК» принадлежит K2 Tech, развивается как K2 Cloud (cloud.croc.ru → k2.cloud). EKS-совместимый API.

- **Control plane:** **бесплатный**, мастера HA по 3 зонам без доплаты [точно].
- **Worker-ноды:** обычные compute-инстансы; **рублёвые ставки не опубликованы** (только в авторизованной консоли / API GetTariff). 4 vCPU/8 ГБ ≈ 4 000–7 000 ₽ [оценка]; 8 vCPU/16 ГБ ≈ 8 000–14 000 ₽ [оценка].
- **SLA:** 99,95% (фактически ~99,982%). Компенсация: сервис-кредиты «до 100% месячной стоимости» [частично подтверждено]. Реакция на критичные ≤10–15 мин, персональный менеджер.
- **Балансировщик (NLB):** бесплатно. ALB бесплатно до 01.04.2026, далее по тарифу (не опубл.).
- **Egress, диски, IP, S3:** ставки не опубликованы [оценка S3 ~1,3–2,5 ₽/ГБ].
- **k8s/k3s:** vanilla k8s через EKS-совместимый API (v1.30–1.33); k3s нет.
- **Прод-HA итог:** ≈ **16 000–19 000 ₽/мес** [оценка — для точного бюджета нужен запрос в sales: info@k2.cloud].
