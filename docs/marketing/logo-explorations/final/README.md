# KISS PM logo assets

Финальное направление: синяя app-иконка с белой K-галочкой. Знак должен читаться как `K`, подтверждение и примененное изменение.

## Файлы

- `kiss-pm-app-icon-1024.png` — основной raster source для app/social. Каноническая форма знака.
- `kiss-pm-app-icon-512.png` — web/social preview.
- `kiss-pm-app-icon-256.png` — интерфейсные превью.
- `kiss-pm-app-icon-32.png` — favicon/sidebar micro.
- `kiss-pm-wordmark.svg` / `kiss-pm-wordmark.png` — текстовый знак.
- `kiss-pm-lockup-horizontal.svg` / `kiss-pm-lockup-horizontal.png` — горизонтальная связка icon + wordmark. SVG использует canonical raster icon внутри, чтобы форма знака не расходилась.

У app-icon PNG прозрачные углы; белая K-галочка внутри сохранена из выбранного raster-кандидата, без ручной перерисовки.
