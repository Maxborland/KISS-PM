## Как пользоваться

Перед фиксом поверхности открой соответствующий рендер как визуальную цель. Сквозной вывод (принцип 1): число → `число+спарклайн, правый край`; категория → `цветной чип/точка`; человек → `аватар`; лог → `таймлайн+аватары в шторке`; редкое действие → `поповер/меню`; неактивное → `приглушить`.
